Sample Project for TypeScript + React
========================================

Requirements
------------------------------

* [Java Runtime Environment](http://www.oracle.com/technetwork/java/javase/downloads/index.html).
    * Used by Gradle (build system).

Build
------------------------------

To build, execute:

```
$ ./gradlew assemble
```

To see all tasks:

```
$ ./gradlew tasks
```
